TranExcel

TranExcel is the macroscopic transport planning web application which helps DPT's planner to find the existing and forecasting traffic volume on transport network base on DPT's GIS database and traffic survey data.

Trip Generation

To predict the number of trips originating in or destined (each mode) for a particular traffic analysis zone.

Go TG Module
Trip Distribution

To predict the number of trips that occur between each origin zone and each destination zone.

Go TD Module
Trip Assignment

To concern the selection of routes between origins and destinations and calculate the traffic volume.

Go TA Module
Network Builder

To display the transport network data in graphic form.

Go NB Module
Shortest Path

To find the shortest path all of traffic analysis zones.

Go SP Module
Matrix Estimation

To estimate the OD trip table from traffic count data.

Go ME Module
EE and IE/EI Estimator

To estimate the EE and EI/IE trip from traffic count data.

Go EE,IE/EI Module
Building Area Usage Formatting

To change the building area usage data format from ArcGIS to TranExcel.

Go BU Formatting Module
